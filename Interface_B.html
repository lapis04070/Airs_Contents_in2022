インターフェース Bパターン

配列の活用

1つの配列で2つのオブジェクトを管理できるインターフェースを使って、2つのオブジェクトを交互に表示してみます。

step.1

始めに、ChickenタブとOwlタブを作成し、クラスの定義を行います。(ChickenとOwlの作成は省きます。)

step.2-1

次に、OwlオブジェクトとChickenオブジェクトの2つを管理するためにインターフェースを定義し、これをOwlオブジェクトとChickenオブジェクトに実装します。インターフェースの名前を「Bird」にするため、「Bird」タブを追加します。

 Bird[]birds = new Bird[11];

void setup(){
size(770,120); //ウインドウを描く
background(176, 204, 226);
for(int i = 0; i < birds.length; i++){
if((i % 2) == 0){ //iを2で割った時の余りが0か1で判断できる。
birds[i] = new Owl(35 + i * 70, 110); //上の方が先に表示される。今回だとOwlが先
} else { //余りが0ではない場合は、下のbird[i]が実行
birds[i] = new Chicken(35 + i * 70, 110);//逆でも問題ない。
}
}
}
void draw(){
for(int i = 0; i < birds.length; i++){
birds[i].display();

}

今回は、合計11匹のOwlとChickenを表示させましたが、数を増やしたり減らしたりすることができます。
