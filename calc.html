計算

変数と計算

コードには+や*といった記号が多く使われています。

これらは演算子と呼ばれ、2つの数値の間に置かれ、計算式を表します。


基本的な演算子

演算子 説明
+ 足す(加算)
- 引く(減算)
* かける(乗算)
/ 割る(除算)
= 代入

Processingでは演算子の優先順位があります。
乗算、除算のほうが加算、減算よりも優先されるルールがあります。
また、カッコ()で囲んだ中は最も高い優先順位で処理されます。
頻繁に使用する式では簡単にかけるよう、以下のような短縮形も用意されています。

x += 10;//x = x + 10と同じ
y -= 15;//y = y - 15と同じ
x++;//x = x + 1と同じ
y--;//y = y - 1と同じ
