クラスの継承

Processingではクラスは他のクラスのフィールドとメソッドを継承することができます。
継承されるクラスを親クラス(Super Class)継承するクラスを子クラス(Sub Class)と言います。
親クラスは今まで通りクラス作成、子クラスは新しいタブにclass 子クラス名 extends 親クラス名を記述して定義します。

 class SubJitterBug extends JitterBug {
SubJitterBug(float tempX, float tempY, int tempDiameter) {
super(tempX, tempY, tempDiameter);
}
}

この例ではまず1行目に子クラスの定義、2行目でコンストラクタの記述を行っています。
コンストラクタの中身のsuperは親クラスのコンストラクタを呼び出す事ができる特殊なメソッドです。
こうすることで親クラスと同じ状態を持った子クラスを作ることができます。
ここから親クラスにはないフィールドやメソッドを追加することでクラス拡張をすることができます。
