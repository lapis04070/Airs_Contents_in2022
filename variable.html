4章 変数

変数

変数の作成

変数とは数値や文字といった値を入れるための箱のようなものです。

変数には特定のデータ型があり、代入する値の種類によってデータ型を使い分けます。


代表的なデータ型

名前 扱うデータ
int 整数
float 浮動小数点数
char 文字(1文字)
String 文字列

変数を作成するにはデータ型、変数名になります。


1
2
 int x;xをint型変数として宣言
x = 12;//xに値を代入する

この例では、変数xをint型として宣言。その後、xに12という整数と代入しています。
また、変数名の前にデータ型を書くのは最初の宣言のとき1回のみです。同じ名前の変数を2回宣言するとエラーになるので注意が必要です。
