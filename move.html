細かい動きを表現するには小数点の値を持つ数値を格納できるfloat型を使用します。

 float x = 40;
float speed = 0.5;

void setup(){
size(240,120);
}

void draw() {
background(0);
x += speed;//xに加算
arc(x,60,40,40,0.52,5.76);
}

ここでは、円の中心のx座標をfloat型の変数xとして右へと移動するプログラムです。
float型の変数xにdraw()関数の実行に合わせて変数speedの値が追加されていきます。
これにより円の中心の値が増加していき、図形を移動させる事ができます。
