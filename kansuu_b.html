void draw(){

描いたフクロウをもう1羽描くと、コードの長さが倍になってしまいます。
そこで、フクロウのコードに関数を導入します。
フクロウを描くコードを関数に入れてしまえば、そのコードはもうプログラム中に一度しか現れません。

 void setup(){
size(480,120); //ウインドウを描く
}

void draw(){
background(176,204,226);
owl(110,110);
owl(180,110);
}

void owl(int x,int y){
pushMatrix(); //現在の座標系を保存
translate(x,y);
stroke(138,138,125);
strokeWeight(70);
line(0,-35,0,-65); //胴体
noStroke();
fill(255);
ellipse(-17.5,-65,35,35); //左目のまわり
ellipse(17.5,-65,35,35); //右目のまわり
arc(0,-65,70,70,0,PI); //あご
fill(51,51,30);
ellipse(-14,-65,8,8); //左目
ellipse(14,-65,8,8); //右目
quad(0,-58,4,-51,0,-44,-4,-51); //くちばし
popMatrix(); //元の座標系を取り出す
}

owl()関数にフクロウを描くコードをまとめることで、コードの長さを短くします。
フクロウのコードはdraw()から2回呼び出されるので、2回実行されます。
呼び出す際に、(x座標,y座標)の2つのパラメータを関数に渡すことで、2羽のフクロウを異なる位置に表示させます。
