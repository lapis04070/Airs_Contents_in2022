14章 インターフェース

インターフェース

インターフェース
複数のオブジェクトを管理するために、第11章で配列を使う例を上げましたが、配列は同じ型のオブジェクトしか扱うことができません。
一つの配列で二つのオブジェクトの両方を管理できるのが、インターフェースの概念です。

インターフェースの定義

作成した2つのオブジェクトの両方を管理するためにインターフェースを定義し、これを2つのオブジェクトに実装します。
1
2
3
 interface インターフェース名{ //作成したタブの名前を記述する
void display();//displayメソッドの型と名前の部分のみ記述し、本体は記述しません。
}
この操作は、インターフェースの抽出と言います。
