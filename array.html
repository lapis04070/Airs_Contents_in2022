変数から配列へ

配列の作り方

配列とは同じデータ型の値を複数格納し、一つの変数にまとめて使用することができます。

また、格納された値は何番目かを指定して取り出すことができるのも特徴です。

1
2
3
 int[] x;//配列を宣言

x = new int[2];//配列の生成

上記の例ではまず、配列となるxを宣言しています。データ型の後ろに[]をつけて宣言をします。
その後、2個の整数からなる配列xを作ります。
配列を作るときには[]付きのデータ型、変数名、=、new、データ型、[格納する数]の書式です。

x[0] = 12;//1つ目の値を代入
x[1] = 2;//2つ目の値を代入
}

配列の中身の項目は要素と呼ばれ、各要素にはインデックスと呼ばれる位置を示すものがあります。インデックスは0から数えます。
5行目ではインデックスの0番目の要素に12を、6行目ではインデックスの1番目の要素に2を代入しています。
また、以下のように一括で作る方法もあります。
