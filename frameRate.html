フレームレート

フレームの仕組み

Processingではdraw()内のコードを毎秒60回呼び出し実行しています。

この呼び出される画面をフレームと呼び、1秒間に何回画面が呼び出されるのかを表すのがフレームレートです。

1
2
3
 void draw() {
println(frameRate);
}

上記の例のdraw()関数は毎秒60回連続で実行されています。frameRate変数は1秒間に何回画面が更新されるかを知ることができます。
また、frameRate()関数を使うとプログラムの実行スピードを変更することができます。

 void setup() {
frameRate(30);//毎秒30フレーム
frameRate(12);//毎秒12フレーム
frameRate(2);//毎秒2フレーム
frameRate(0.5);//2秒ごとに1フレーム
}

void draw() {
println(frameRate);
}

このように直接指定することでフレームレートを調整する事ができます。
