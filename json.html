JSON形式の読み込み

JSON(JavaScript Object Notation)はデータの交換に適したフォーマットになっています。

実例としてタイトル、監督、公開年、レーティングのデータを持った映画の情報を使えるようにします。

 {
"title": "Alphaville",
"director": "Jean-Luc Godard",
"year": 1964,
"rating": 7.2
}

このパターンを繰り返すことで、たくさんの映画の情報を記録することができます。
CSV形式との違いとしてCSVの方が少ない文字数で表現できる一方、JSON形式ではラベルがあるため人にとって認識しやすい形になっています。
以下はJSON形式のファイルを読み込む例です。

 JSONObject film;

void setup(){
film = loadJSONObject("film.json");
String title = film.getString("title");
String dir = film.getString("director");
int year = film.getInt("year");
float rating = film.getFloat("rating");
println(title + "by" + dir + ", " + year);
println("Rating: " + rating);
}

今回はfilm.jsonという名前の映画の情報が入ったファイルを読み込みました。
今回の結果はコンソールにて以下のように出力されます。

2
 AlphavillebyJean-Luc Godard, 1964
Rating: 7.2
