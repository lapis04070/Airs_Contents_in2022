6章 移動、回転、収縮

座標の移動

座標移動

translate()関数は、座標系を移動させる関数です。

1
2
3
4
 void draw() {
translate(mouseX,mouseY);
rect(0,0,30,30);
}

Processingでは初期状態は座標(0,0)は画面左上に設定されています。
translate()関数では座標(0,0)の値を変える事ができるため、今回のプログラムではマウス座標が(0,0)になるよう設定されています。
rectは座標(0,0)を基準に描画されるため、マウスが移動すると図形も移動します。

次に、rotate()関数は座標系を回転させる関数です。

1
2
3
4
 void draw() {
rotate(mouseX / 100.0);
rect(0,0,160,20);
}

図形を回転させるにはrotate()関数を使い、回転角を決定します。今回はマウスのX座標を角度に変換しています。
rotate()関数を使うことで画面内の座標系が回転するため、図形が回転移動させることができます。

最後にscale()関数は座標系を伸縮させる事ができます。

1
2
3
4
 void draw() {
scale(1.5);
rect(0,0,160,20);
}

図形を伸縮させるにはscale()関数を使います。画面のすべての座標が値に応じて拡大、縮小されます。
例として今回では1.5なのですべてが150%の大きさに拡大されます。
