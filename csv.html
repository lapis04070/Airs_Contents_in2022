CSV形式の読み込み

表形式のデータは、カンマやタブで区切られたプレーンテキストファイルとして保存されていることがあります。
そのようなファイルをCSV(comma-separated values)ファイルといい、拡張子は".csv"です。

例としてmediaファイル内のortiz.csvのデータをProcessingに読み込ませます。

 Table stats;

void setup(){
stats = loadTable("ortiz.csv");
for (int i = 0; i < stats.getRowCount(); i++){
int year = stats.getInt(i, 0);
int homeRuns = stats.getInt(i, 1);
int rbi = stats.getInt(i, 2);
float average = stats.getFloat(i, 3);
println(year, homeRuns, rbi, average);
}
}

今回はortiz.csvファイルを読み込みました。
結果、csv内のデータがProcessingのコンソール内に表示されます。
