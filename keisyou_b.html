継承 Bパターン

継承

親クラスを継承し、子クラスを作ることができます。親クラスを継承することによって、子クラスは親クラスのフィールドとメソッドをすべて引き継ぐことができます。

実際に、親クラスと子クラスを作成し継承してみましょう。
今回は、フクロウ(Owl)を親クラスにし継承していきます。

step.1

始めに親クラスのフクロウを作成します。
タブの名前を「Owl」にします。

 class Owl{
int x;
int y;
Owl(int tempX,int tempY){
x = tempX;
y = tempY;
}

void display(){
pushMatrix();
translate(x,y);
stroke(138,138,125);
strokeWeight(70);
line(0,-35,0,-65);
noStroke();
fill(255);
ellipse(-17.5,-65,35,35);
ellipse(17.5,-65,35,35);
arc(0,-65,70,70,0,PI);
fill(51,51,30);
ellipse(-14,-65,8,8);
ellipse(14,-65,8,8);
quad(0,-58,4,-51,0,-44,-4,-51);
popMatrix();
}
}

step.2

次に、子クラスを定義します。タブ名は「SubOwl」にします。
SubOwlのクラス名を入力する際に、親クラスを継承していることを示すために、extends Owlをクラス名の後に記述します。
1
2
3
4
5
 class SubOwl extends Owl{
SubOwl(int tempX, int tempY){
super(tempX,tempY);
}
}

よって、親クラスと同じ状態やメソッドを持った、子クラスが完成しました。
継承したことで、SubOwlの中身は、少ないコードになっております。
